== Description ==

Turns all mp3 links into a playlist using Yahoo Media Player API. To use this plugin you should agree to the <a href="http://info.yahoo.com/legal/us/yahoo/utos/utos-173.html">Yahoo! Terms of service</a>.

== Installation == 

1. Upload the plugin folder to /wp-content/plugins
2. Activate the plugin from Appearance->Plugins
